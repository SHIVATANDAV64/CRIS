## Core Mechanism
AF-Retriever is a modular pipeline for multi-hop question answering over Semi-Structured Knowledge Bases (SKBs), which combine structured data (knowledge graphs, tables) with linked unstructured content (natural language documents). The system operates through four constraint-driven retrieval steps followed by four processing steps:

1. **LLM-based Constraint Extraction**: Uses large language models to parse queries and extract entity attributes and relational constraints
2. **Vector Similarity Search**: Ranks extracted entities and final answers using embedding similarity
3. **Incremental Scope Expansion**: Iteratively expands the candidate pool based on how well candidates satisfy constraints—similar to an optical autofocus continuously adjusting focus
4. **Hybrid Retrieval Strategy**: Combines structural and textual retrieval to reduce error susceptibility

The pipeline delivers a configurable amount of answer candidates while "constantly adjusting the focus" based on constraint satisfaction.

## Key Insight
The core insight is the **autofocus analogy** for information retrieval: instead of a single-pass retrieval, the system iteratively adjusts its search scope by expanding candidates based on how well they fulfill the extracted relational constraints. This incremental expansion allows the system to prepare a configurable set of high-quality candidates for final reranking, balancing precision (through constraint filtering) with recall (through scope expansion). The use of exchangeable LLMs as both constraint extractors and rerankers provides flexibility while maintaining strong zero-shot and one-shot performance.

## Domain-Blind Abstraction
A system that combines multiple retrieval pathways (structural and content-based) with iterative scope expansion to find answers requiring multi-step reasoning. It uses a configurable pipeline where initial broad retrieval is progressively narrowed through constraint satisfaction evaluation, then expanded again for candidate diversity—analogous to a focusing mechanism that adjusts depth-of-field to capture both fine details and broader context simultaneously.
