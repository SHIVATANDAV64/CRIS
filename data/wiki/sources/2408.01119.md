## Core Mechanism
Task Prompt Vectors are created by computing the element-wise difference between the weights of a tuned soft-prompt and its original random initialization. Mathematically: **TV = P_tuned - P_random**, where TV is the Task Prompt Vector. This vector captures the "delta" or adjustment needed to transform a random prompt into a task-specific one. These vectors can then be added to new random initializations to initialize prompt tuning for similar tasks, or combined through arithmetic operations (addition, subtraction) to create hybrid task behaviors.

## Key Insight
The key insight is that the **difference between a trained soft-prompt and its random initialization encodes task-specific knowledge** in a way that is independent of the specific random seed used. This means the Task Prompt Vector represents the "essence" of task adaptation rather than artifacts of a particular initialization. Because this vector is initialization-independent, it can be transferred across different prompt initializations and even across different model architectures.

## Domain-Blind Abstraction
When you take a trained parameter vector and subtract its initial random state, you get a "direction of improvement" - a transformation vector that moves from random initialization to task competence. This transformation vector is reusable because it captures the underlying pattern of adaptation rather than the specific starting point. By adding this transformation to new random starting points, you can accelerate learning on related tasks.

## Cross-Domain Potential
- **Computer Vision**: Apply similar vector arithmetic to learned visual prompts for image classification or generation tasks
- **Reinforcement Learning**: Extract policy "delta" vectors from trained agents to initialize learning on related environments
- **Federated Learning**: Use parameter difference vectors for efficient knowledge transfer between distributed clients
- **Multimodal Models**: Transfer prompt vectors across modalities (e.g., language-to-image prompt initialization)
