## Core Mechanism
TRACE (Temporal Representation and Classification of Exploits) is a vendor-conditioned contrastive learning framework built on CySecBERT (a cybersecurity-domain pretrained language model). The framework jointly optimizes two objectives: (1) organizational target classification (predicting which of 7 organization categories a threat targets) and (2) vendor-coherent representation learning via contrastive learning. The "vendor-conditioning" aspect pulls together representations from the same vendor while pushing apart representations from different vendors, creating embeddings that capture vendor-specific patterns. The framework is evaluated under temporal distribution shift—training on earlier data and testing on later temporal splits to assess robustness over time.

## Key Insight
The key insight is that incorporating vendor-specific structure into the representation learning process through contrastive learning improves both classification accuracy and temporal robustness. By learning representations that are coherent within vendors (attack patterns, exploit styles), the model generalizes better to new temporal distributions where exploit patterns evolve. The large-scale multi-source dataset (352,866 posts from 9 sources over 30 years) enables learning robust patterns that transfer across time periods.
