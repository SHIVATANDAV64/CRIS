## Core Mechanism
ChatSR treats scientific data (numerical sequences, measurements, experimental results) as a new modality analogous to images or text. It uses specially designed **encoders** to convert scientific data into embeddings that can be processed by a large language model. The system employs **modality alignment mechanisms** to map scientific data into the same representation space used by the LLM for text processing.

Once the scientific data is encoded, ChatSR leverages the LLM's reasoning capabilities to generate mathematical formulas. Unlike traditional symbolic regression that only fits curves to data, ChatSR accepts **user-specified prior constraints** (periodicity, symmetry, monotonicity) and ensures generated formulas satisfy both:
1. Accurate fit to observed data points
2. Conformance to domain-specific priors (e.g., physical laws, mathematical properties)

## Key Insight
The core insight is that scientific data can be treated as a first-class modality, similar to how images were integrated into LLMs. By encoding scientific data into the LLM's representation space, the model can leverage its pre-trained knowledge about mathematical structures and physical laws to guide formula discovery. The LLM acts as a "knowledgeable scientist" that can reason about what form a formula should take based on both the data patterns AND prior domain knowledge—achieving something traditional curve-fitting cannot.

## Domain-Blind Abstraction
A system that takes numerical observations as input, transforms them into a unified representation, then uses a trained reasoning model to generate mathematical expressions that satisfy both: (1) fitting the observed data points within error bounds, and (2) satisfying user-specified structural constraints on the expression's form.
